# CountdownView
使用一个TextView来实现丰富的倒计时功能

倒计时显示，是Android开发中常常遇到的一个需求，电商平台尤甚。但是Android原生的倒计时控件，扩展性和可定制性太差。目前看到的各个APP中，大部分都是采用一个继承自ViewGroup的组合控件来实现。这无形中就增加了一层View的嵌套。基于此方面的考虑，本项目尝试直接通过TextView来实现倒计时。包括对倒计时样式的定制。
